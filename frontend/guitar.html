Virtual Guitar 연주

선택된 코드: 없음

키보드 설정

코드 모드 키 설명

  • 1: Major (메이저)
  • 2: 7 (세븐)
  • 3: m (마이너)
  • 4: m7 (마이너 세븐)
  • 5: m7 (메이저 세븐)
  • 6: Barre Chord (배레 코드)

Barre Chord는 한 손가락으로 여러 현을 동시에 누르는 코드 형태입니다. 이를 통해 다양한 코드를 같은 형태로 이동시키며 연주할 수 있습니다.

코드 모드 키 조합 예시

예를 들어, C와 1을 조합하면 C Major 코드가 됩니다.
C와 2를 조합하면 C7 코드,
C와 3을 조합하면 Cm 코드,
C와 4를 조합하면 Cm7 코드,
C와 5를 조합하면 CM7 코드,
C와 6를 조합하면 C Barre Chord가 됩니다.
